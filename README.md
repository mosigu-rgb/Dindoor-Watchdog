# Project Dindoor: AI Infrastructure & Resilience Watchdog

This project focuses on the intersection of **Physical AI Safety** and **Strategic Infrastructure Resilience**. It provides a technical solution for monitoring GPU cluster health while analyzing large-scale vulnerabilities in critical data center hubs.

---

## 🛡️ Core Concept
While most AI safety research focuses on alignment and software, **Project Dindoor** addresses the physical layer. The core objective is to detect anomalies in GPU clusters that could indicate coordinated load-fluctuation attacks or grid-level instability.
