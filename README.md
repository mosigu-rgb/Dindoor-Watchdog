# Dindoor-Watchdog

Advanced Threat Detection System for critical server infrastructure. 

## Project Evolution
- **v1.0**: Basic IP-based filtering (Rule-based logic)
- **v2.0**: Intelligent Weighted Risk Scoring & Statistical Analysis

## Current Features
- **Weighted Scoring**: Evaluates CPU load, VPN status, and packet sizes.
- **Automated Reporting**: Generates security audits with threat detection rates.
- **Scalable Design**: Built to monitor high-density GPU clusters (Virginia Region).
